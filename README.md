# customer_behaviour_analysis
Customer Behaviour Analysis project using Python, SQL, and Power BI to uncover insights from ~3,900 transactions. Includes data cleaning, EDA, customer segmentation, and business-focused analysis. Features an interactive dashboard and actionable recommendations to improve marketing, retention, and revenue strategies.
# Customer Behaviour Analysis

## Overview  
This project analyzes customer purchasing behavior using **Python, SQL, and Power BI** to generate actionable business insights. The objective is to understand customer preferences, spending patterns, and segmentation to support data-driven decision-making.

## Dataset  
The dataset contains approximately 3,900 customer transactions, including:  
- **Demographics:** Age, Gender, Location, Subscription Status  
- **Transaction details:** Product category, items purchased, purchase amount  
- **Behavioral data:** Purchase frequency, previous purchases, discounts  
- **Additional attributes:** Shipping type, ratings, season, size, color  

## Methodology  

### Data Preprocessing and EDA (Python)  
- Cleaned and handled missing values  
- Standardized data formats  
- Performed exploratory data analysis  
- Created derived features such as age groups and purchase frequency  

### Data Analysis (SQL)  
- Revenue analysis across customer segments  
- Identification of high-spending customers using discounts  
- Top-rated products based on customer reviews  
- Comparison of shipping types and spending behavior  
- Customer segmentation into New, Returning, and Loyal groups  
- Analysis of subscription impact on revenue and repeat purchases  

### Data Visualization (Power BI)  
- Built an interactive dashboard to present key insights  
- Visualized trends in revenue, customer segments, and product performance  

## Key Insights  
- Subscription-based customers contribute significantly to revenue  
- Repeat customers show higher engagement and spending  
- Discounts influence purchasing behavior but require optimization  
- Certain products consistently receive higher ratings and demand  

## Business Recommendations  
- Strengthen subscription programs to improve retention  
- Implement loyalty strategies for repeat customers  
- Optimize discount policies to balance revenue and profitability  
- Focus marketing efforts on high-value customer segments  

## Tools and Technologies  
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL  
- Power BI  

## Conclusion  
This project demonstrates how data analysis techniques can be used to extract meaningful insights from customer data and support strategic business decisions.
